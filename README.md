# Fresh Tomatoes

[로튼 토마토](https://www.rottentomatoes.com/)의 컨셉을 클론해 만든 영화/TV프로그램 정보 제공 어플리케이션 토이 프로젝트입니다.

## Features

- ISR, SSR
- Infinite scrolling
- Dynamic routing
- Infinite carousel, mutiple carousel
- Image lazy loading
- Responsive design

## Live Demo

[Link](https://fresh-tomatoes.vercel.app/)

## Tech Stacks

- Next.js
- TypeScript
- Emotion.js
- Vercel

## Getting Started

1. Clone this repository

```
git clone https://github.com/iwaskorean/fresh-tomatoes.git
```

2. Install

```
cd fresh-tomatoes
yarn
```

3. start

```
yarn build
yarn start
```
